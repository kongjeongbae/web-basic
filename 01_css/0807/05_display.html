1. Block 레벨 element

- h계열의 태그

- p 태그

- div 태그
- form 태그
  • 항상 새로운 라인에서 시작합니다.
  • 화면 크기 전체 폭을 차지합니다. (width: 100%)
  • inline 요소를 포함할 수 있다.
  • ul, li 도 block 요소

2. Inline 레벨 element

inline는 한 줄에 표시되며, width, height, margin_top, bottom 사용 불가

- a 태그 - span 태그
  • 자신의 컨텐츠만큼 자리를 차지합니다.
  • inline은 margin-bottom, margin-top을 못준다.

3. Inline-block

  1. block과 inline요소의 특징을 모두 가짐
  2. inline이지만 block에서 활용되는 속성을 활용할 수 있다.
  3. width, height, margin-top, margin-bottom 값을 적용할 수 있다.
